# OpenGLRenderer
Realtime 3D Model Viewer for Windows using OpenGL 4<br/>
<br/>
Mesh formats: .gltf, .glb, .obj<br/>
Image formats: .png, .hdr, .tga, .jpg, .psd<br/>
